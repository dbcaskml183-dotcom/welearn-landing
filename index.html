您可能想问的

这个工具安全吗？会被学校发现吗？
工具在本地运行，所有操作模拟真实浏览器行为（随机延迟、鼠标轨迹模拟），与人工操作无异。数据仅保存在本地，不会上传你的个人信息。但我们建议合理使用，不要在同一门课上过度刷时长。
需要什么样的电脑配置？
Windows 10/11 即可，集成显卡轻薄本完全够用。所有 AI 计算在云端完成，本地只需要能运行 Python 和浏览器。工具首次启动会自动安装所需依赖。
DeepSeek API Key 怎么获取？
打开 DeepSeek 官网（platform.deepseek.com）注册账号 → 创建 API Key → 复制粘贴到工具中即可。首次注册赠送 500 万 tokens，足够用一学期。
支持 Mac 或 Linux 吗？
当前版本仅支持 Windows 10/11。Mac 和 Linux 版正在开发中，预计下学期上线。你可以关注我们的更新通知。
